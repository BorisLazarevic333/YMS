# Yard Management System (YMS)

Modern MERN stack Yard Management System for real-time trailer and container operations.

---

## Features

- User Authentication
- Guest Mode
- Ingate / Outgate Operations
- Trailer Pool Management
- Analytics Dashboard
- History Tracking
- Filters & Pagination
- Responsive Design
- Admin Controls
- Modern UI with Tailwind CSS

---

## Tech Stack

Frontend:

- React
- Tailwind CSS
- Framer Motion
- GSAP

Backend:

- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
